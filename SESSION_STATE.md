### TQL — TRUTH QUALITY ASSESSMENT

Mandatory per AI-006. Every case, every edition. Exactly four items in this order.

**Item 1 — Key Assumption**
The single most load-bearing assumption the case analysis depends on.
Must state: the assumption AND the counter-assumption.

**Item 2 — Fragile Fact**
The single fact that, if wrong, would most materially change the hypothesis set.
Must state: the source tier AND the direction of impact if wrong.

**Item 3 — High-Impact Uncertainty**
The unknown that, if resolved either way, would produce the largest probability shift.
Must state: the shift directionally (e.g. “H-A1 falls to 40-50% if X”).

**Item 4 — Tier 4 Dependency Test**
Remove all Tier 4 sources. Does the dominant hypothesis still hold?
Answer must be YES or NO with one sentence of justification.

Omission of any item = deviation audit failure. Do not build PDF until all four items present in all five cases.

-----

### GATES — DEFINITIONS IN FULL

**Gate 0 — Source Admission**
A source is admitted to analysis only if it can be assigned a tier (1–4) and a named incentive assessment. Anonymous sources require explicit Gate 0.4 rating. No source enters a fact table without both fields populated.

**Gate 0.3 — Absence Confirmation**
A claim that something did not happen must be confirmed across a minimum of three mandatory feeds. Mandatory feeds for kinetic events: CNN, NBC, Al Jazeera, Naval News. Label confirmed absences as [FACT — absence observable]. If fewer than three feeds checked, label as [UNVERIFIED ABSENCE].

**Gate 0.4 — Confidence Rating**
